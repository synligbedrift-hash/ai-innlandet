# AI Innlandet

Nettside for AI Innlandet – kunstig intelligens for bedrifter i Innlandet.

## Teknisk
- Statisk HTML/CSS
- Hostes på Netlify
- Kildekode på GitHub
- Leadskjema via Netlify Forms

## Deploy
Automatisk via Netlify når endringer pushes til `main`-branch på GitHub.
